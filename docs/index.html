Environmental sound conversion from vocal imitations and sound event labels
　　

This is a demonstration of environmental sound conversion from vocal imitations. We conducted environmental sound conversion using two proposed methods, one comparison method, and ground truth as follows:

    　
  • Reconstructed sound
    • This is a reconstructed sound from the ground-truth mel-spectrogram by the neural vocoder.
    　
  • Synthesis method using sound event labels (Label) (Baseline)
    • This method is uses a sound event labels as an input.
    　
  • Synthesis method using vocal imitations (Vocal) (Proposed)
    • This method is uses a vocal imitations as an input.

  • Synthesis method using vocal imitations and sound event labels (Vocal and label) (Proposed)
    • This method is uses a vocal imitations and sound event labels as an input.
